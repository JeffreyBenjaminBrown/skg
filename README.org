* to understand the code
The graph uses [[./schema.tql][this TypeDB schema]]. The server is written in Rust, which makes TypeDB queries. Clients could be written for other editors; so far I've written one, for Emacs, in Emacs Lisp.

The API, the view metadata format, and the .skg file format are described in [[./api-and-formats.md][api-and-formats.md]].
