- save extraction owns interpretation of user-authored direct
  =AsSubscribee= child lists as subscriber visibility edits;
- saved-view completion preserves definitive subscribee-as-such
  child lists until save extraction can interpret them;
- collateral-view completion and diff-mode completion may regenerate
  branches from graph/diff facts, because they are not the saved
  buffer's pending user edit surface;
- generated display scaffolds may be reconciled during rerender, but
  save extraction must continue treating them as display-only;
- diff-mode phantom/removed nodes are display state and must not become
  hide/unhide signal.
