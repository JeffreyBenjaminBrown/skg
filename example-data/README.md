This is how you might set up your data
if you kept separate public and private data,
and if you also used Fred's public data.
Each of those folders would contain .skg files.
Hopefully, each will be a git repo.
The collection of them would be 'your graph'.
Even though you never change the data in Fred,
you can change the way you experience it --
look for the terms 'lens', 'subscribe', 'override'
or (soon) 'fork' in the documentation for details about that.
