* Refactor: Replace EffectOnParent with parent_ignores: bool

** Overview

Replace ~EffectOnParent~ enum (Content, Subscribee, ParentIgnores, HiddenFromSubscribees) with a single boolean:
: parent_ignores: bool  // defaults to false

The other discriminators are inferred from tree position:
- Parent is ~SubscribeeCol~ -> was Subscribee
- Parent is ~HiddenInSubscribeeCol~ or ~HiddenOutsideOfSubscribeeCol~ -> was HiddenFromSubscribees
- Otherwise -> was Content (when ~parent_ignores = false~)

1. Delete ~EffectOnParent~ enum entirely (including REPRS_IN_CLIENT and methods)
2. In ~TrueNode~ struct: replace ~effect_on_parent: EffectOnParent~ with ~parent_ignores: bool~
3. Update ~TrueNode::default()~: set ~parent_ignores: false~
4. Update ~mk_orgnode~: change parameter to ~parent_ignores: bool~
5. Update ~mk_definitive_orgnode~: pass ~false~
6. Update ~mk_indefinitive_orgnode~: change parameter to ~parent_ignores: bool~

*** Phase 3: Parsing (server/serve/parse_metadata_sexp.rs)

2. Add ~parent_ignores: bool~ field to ~OrgnodeCode~ struct.

3. Delete ~Interp::to_effect_on_parent~ method entirely.

4. Update ~parse_code_sexp~: when parsing ~(interp X)~:
   - If X is a scaffold string -> use ~Scaffold::from_repr_in_client~

   - Otherwise -> error

5. Update ~from_parsed~:
